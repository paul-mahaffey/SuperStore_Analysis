# Superstore Sales Performance Analysis

## Tools Used
- Microsoft Excel – data cleaning and validation
- SQL Server – data ingestion, validation, and business analysis
- Power BI – dashboard creation and data visualization

## Business Objective
Analyze retail sales performance to identify trends in revenue, profitability, discount impact, and operational efficiency.

## Key Business Questions
- Which regions generate the most revenue and profit?
- Which product categories and sub-categories drive or drain profitability?
- How do discounts affect profit margins?
- Which shipping methods are most operationally efficient?

## Process
1. Cleaned and validated raw Superstore sales data in Excel.
2. Imported the dataset into SQL Server and performed data validation checks.
3. Wrote SQL queries to analyze regional performance, product profitability, discount impact, and shipping efficiency.
4. Built an interactive Power BI dashboard with KPI cards, trend analysis, and filters for business decision-makers.

## Outcome
Delivered an executive-level Power BI dashboard that provides insights into sales performance, highlights underperforming categories, and identifies profit leakage due to excessive discounting.
